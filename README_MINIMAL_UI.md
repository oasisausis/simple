# Complete minimal UI redesign

This package redesigns the complete IITB MSc Physics Fresher Guide using the approved navy-and-warm-ivory visual system.

## Upload everything

Replace the corresponding files in the repository and upload all files inside `assets/`.

Main pages:
- index.html
- academics.html
- campus-life.html
- campus-map.html
- seniors.html
- clubs.html
- transport.html
- resources.html
- events.html
- emergency.html
- styles.css
- site.js
- events-data.js

Compatibility redirect pages are also included: courses.html, grades.html, food.html and monsoon.html.

## Design changes
- warm ivory page canvas instead of bright white
- navy navigation and hero panels
- only six restrained accent colours used for recognition
- consistent icons/symbols, card shapes, spacing and typography
- subtle dotted physics pattern on light sections
- stronger contrast and simpler visual hierarchy
- responsive two-column mobile quick-access cards
- existing interactive map, reminders, calculators, filters and senior search preserved
